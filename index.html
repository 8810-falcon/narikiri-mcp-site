narikiri-mcp

AIの口調を着せ替えるMCPペルソナサーバー

npx narikiri-mcp

AIとの対話に、もう少しだけ余白を

毎日AIエージェントと向き合っていると、正確だけど画一的なアウトプットに、ふと疲れを感じる瞬間はありませんか。

試しに口調を変えてみたら、AIへの親しみが増して、同じ回答でも良きパートナーからの言葉として受け取れるようになりました。

ただ、設定ファイルやシステムプロンプトに直接書く方法には課題がありました。コンテキストの圧迫、トークン消費の増加、設定ミスで環境を壊すリスク。

narikiri-mcp は、MCP プロンプトとしてキャラクターを配信することで、これらをクリーンに解決します。プロジェクトを汚さず、必要な時だけ口調を切り替えられます。

組み込み8キャラに加え、会話しながらオリジナルキャラも作れます

会話するだけでオリジナルキャラが完成

「キャラクター作成」から質問に答えるだけで、あなただけのAIが生まれます。

使い方

1

セットアップ

ターミナルで以下を実行するだけ。対話式で設定が完了します。

npx narikiri-mcp

全プロジェクト共通 or プロジェクト単位を選べます。Node.js が必要です。

2

クライアントを起動

Claude Code や Claude Desktop を起動（または /mcp → Reconnect）すると、自動でサーバーに接続されます。

3

ペルソナを選択

/ を入力してペルソナ一覧から選択。AIの口調が瞬時に切り替わります。

対応クライアント

Claude Code (macOS) Claude Desktop Cursor Windsurf Continue VS Code

MCP Prompts 対応のクライアントが必要です

色付き: 動作確認済み ／ グレー: 未検証（動作する想定）

Tips

!

口調が薄れたら

会話が長くなるとコンテキスト圧縮が発生し、口調が薄れることがあります。その場合はチャット欄に /なんJ民 のようにペルソナ名を入力して選び直してください。

+

オリジナルキャラクター

チャット欄に /キャラクター作成 と入力すると、会話しながらオリジナルのキャラクターを作れます。作ったキャラクターはどのプロジェクトでも使えます。
